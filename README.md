# wjJuanRuiz


El código esta desarrollado en python 3.7.2
Si se presenta el error "OverflowError: Python int too loarge to convert to C long" actualizar la version de python a la 3.7.2 de 32 bits, o la mas reciente disponible

No se encuentran los archivos:
-blogtext, se ignora con el archivo .gitignore, dado su gran peso. Se puede descargar de:
 https://www.kaggle.com/rtatman/blog-authorship-corpus

Se encuentran los archivos:
a-'por tema y edad.py' que cumple con el punto 1 de lo solicitado en el correo de Whale & Jaguar
b-'sentimiento.py' que cumple con el punto 2 de lo solicitado en el correo
c-'correlacion tema edad.py' que cumple con el requisito de un 3er analisis que yo decidiera hacer
 -'temas.csv' como output de a
 -'edades.csv' como output de a
 -'positive-words.txt' como input de b
 -'negative-words.txt. como input de b
 -'sentimiento.csv' como output de b
 -'correlacion.csv' como output de c
 -'correlacion comprobacion.xlsx' como analisis extra
 -'Juan Ruiz W&J.pptx' presentacion resultados y datos de contacto

Citas:
Las palabras con connotacion positiva o negativa se descargaron de aqui
http://www.cs.uic.edu/~liub/FBS/sentiment-analysis.html
y pertenecen al paper de Bing Liu "Sentiment Analysis and Subjectivity"
